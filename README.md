# Pricing Comparison Table

A clean and semantic Pricing Comparison Table built using only HTML. This project demonstrates how to structure pricing plans with tables, headings, lists, buttons, and semantic HTML elements without using CSS or JavaScript.

## Features

Semantic HTML structure
Pricing comparison table
Multiple pricing plans
Feature comparison
Accessible and well-organized layout
Beginner-friendly project
